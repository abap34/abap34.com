Parse Markdown and Render as HTML via almo [1]




Quick Start


$ pip3 install almo-cli

Common options


Argument Description
build Build the target file or directory.
preview Live preview the target file or directory.
-h, --help Show this help message and exit.
-v, --version Show version information.
-t, --template The template file to use.
-s, --style The style file to use.
-o, --output The output file or directory.
-c, --config The configuration file to use. 


$ almo-cli build example/article.md --template example/assets/template.html --style example/assets/style.css -o index.html
